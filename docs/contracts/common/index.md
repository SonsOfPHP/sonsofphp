# Common Contracts

Common Contracts are generic contracts that are not specific to any one package or
project.

## Installation

```shell
composer require sonsofphp/common-contract
```

## Includes Interfaces

* `ArrayableInterface`
* `ComparableInterface`
* `EquatableInterface`
* `JsonableInterface`
* `TryableInterface`
